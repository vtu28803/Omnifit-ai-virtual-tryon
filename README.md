# OmniFit AI – Virtual Try-On

## Overview
OmniFit AI is a prototype mobile application that allows users to visualize how clothes would look on them before purchasing online. The user uploads a selfie and selects a clothing image, and the system generates a virtual try-on preview.

This project was developed as a hackathon prototype to demonstrate how AI-powered virtual fitting rooms can improve online shopping experiences.

---

## Problem
One of the biggest challenges in online fashion shopping is the inability to try clothes before purchasing. This often leads to:

- High return rates
- Poor shopping confidence
- Incorrect size or style choices

---

## Solution
OmniFit AI provides a simple virtual try-on system where users:

1. Upload their selfie
2. Upload a clothing item
3. Generate a virtual preview

This helps users understand how the clothing might look on them before buying.

---

## Features
- Upload selfie image
- Upload clothing image
- Generate virtual try-on preview
- Simple and clean user interface
- Fast demo generation
